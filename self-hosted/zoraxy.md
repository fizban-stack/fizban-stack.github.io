Zoraxy is a general-purpose reverse proxy and forwarding tool with an intuitive web interface for managing your network traffic.

## Key Features

- Reverse proxy with GUI
- SSL certificate management
- Access control and authentication
- Load balancing
- WebSocket support
- Easy configuration

## Why I Use It

Zoraxy provides an intuitive interface for managing reverse proxies. The built-in access control and SSL management make securing services straightforward.
