Vaultwarden is an unofficial Bitwarden-compatible server written in Rust, perfect for self-hosted password management with minimal resource requirements.

## Key Features

- Password vault storage
- Browser extensions
- Mobile apps for iOS and Android
- Two-factor authentication
- Secure sharing
- Self-host with low resources

## Why I Use It

Vaultwarden is my self-hosted password manager. Having complete control over my passwords while maintaining Bitwarden compatibility gives me security and convenience.
