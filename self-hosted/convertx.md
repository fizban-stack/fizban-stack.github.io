ConvertX is a self-hosted universal file converter that supports a wide range of file formats for audio, video, images, and documents.

## Key Features

- Multi-format support
- Batch conversion
- Web-based interface
- Audio and video conversion
- Image format conversion
- Document conversion

## Why I Use It

ConvertX handles all my file conversion needs locally. No need to upload files to online converters or worry about file size limits.
