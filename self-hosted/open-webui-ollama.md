Open WebUI is a feature-rich ChatGPT-like interface that works with local LLMs via Ollama, giving you complete privacy and control.

## Key Features

- ChatGPT-like interface
- Local LLM support via Ollama
- Multiple model support
- Chat history
- Document upload and analysis
- Complete privacy

## Why I Use It

Open WebUI with Ollama gives me a private AI assistant running entirely on my hardware. No data leaves my network and I have complete control over the models.
