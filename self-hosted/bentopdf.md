BentoPDF (Stirling-PDF) is a locally hosted web application that provides a comprehensive suite of PDF manipulation tools.

## Key Features

- PDF merging and splitting
- Page rotation and organization
- Image to PDF conversion
- PDF compression
- Watermarking
- OCR support

## Why I Use It

BentoPDF provides all the PDF tools I need without uploading sensitive documents to third-party services. Everything stays local and private.
