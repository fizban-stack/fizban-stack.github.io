ChangeDetection.io is a self-hosted tool that monitors websites for changes and sends you notifications when updates are detected.

## Key Features

- Visual and text change detection
- XPath and CSS selector support
- Notification integrations
- Screenshot comparisons
- Scheduled monitoring
- API access

## Why I Use It

ChangeDetection helps me track important updates on websites that don't offer RSS feeds or notifications. Perfect for monitoring price changes, stock availability, or content updates.
