Homepage is a highly customizable application dashboard that provides a beautiful interface for accessing all your self-hosted services with built-in service monitoring.

## Key Features

- Service status monitoring
- Widget support for common apps
- Docker integration
- Custom API integrations
- Bookmarks and shortcuts
- Responsive design

## Why I Use It

Homepage provides a beautiful dashboard for accessing all my self-hosted services. The service integration widgets show me at-a-glance status of everything in my homelab.
