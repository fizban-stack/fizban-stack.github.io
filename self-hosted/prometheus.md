Prometheus is an open-source systems monitoring and alerting toolkit with a powerful query language and time-series database.

## Key Features

- Multi-dimensional data model
- Powerful query language (PromQL)
- Time-series database
- Service discovery
- Alerting with Alertmanager
- Extensive exporters

## Why I Use It

Prometheus collects metrics from all my services and infrastructure. Combined with Grafana, it provides comprehensive monitoring and alerting for my entire homelab.
