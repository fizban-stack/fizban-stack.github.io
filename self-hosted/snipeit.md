Snipe-IT is a free, open-source IT asset management system that helps track hardware, software licenses, accessories, consumables, and more.

## Key Features

- Asset tracking and inventory
- Check-in/check-out system
- License management
- Maintenance tracking
- Custom fields and reports
- Email notifications
- QR code generation

## Why I Use It

Snipe-IT keeps track of all my hardware and software assets in one place. It's especially useful for managing equipment across my home lab and knowing what's deployed where.

## Installation Notes

I run Snipe-IT in a Docker container with MySQL as the database backend. The web interface is clean and intuitive, making it easy to manage assets.
