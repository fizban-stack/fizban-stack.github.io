Apache Guacamole is a clientless remote desktop gateway that supports standard protocols like VNC, RDP, and SSH through your web browser.

## Key Features

- Browser-based access
- VNC, RDP, SSH, and Telnet support
- Multi-factor authentication
- Session recording
- File transfer capabilities
- Connection sharing

## Why I Use It

Guacamole lets me access all my servers and desktops through a web browser without installing client software. Perfect for accessing my lab from any device.
