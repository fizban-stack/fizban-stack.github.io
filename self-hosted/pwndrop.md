PwnDrop is a self-deployable file hosting service designed for red teamers, allowing you to easily upload and share files during security assessments.

## Key Features

- File upload and sharing
- Custom download URLs
- Access logging
- File expiration
- Password protection
- Lightweight and portable

## Why I Use It

PwnDrop provides a simple way to host and share files during security assessments. The access logging helps track file downloads during engagements.
