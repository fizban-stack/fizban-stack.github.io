Broken Crystals is a modern vulnerable web application built with React and TypeScript, designed to demonstrate contemporary web application security issues.

## Technology Stack

- Frontend: React with TypeScript
- Backend: Node.js/Express
- Database: MongoDB
- Architecture: Microservices

## Key Vulnerabilities

- SQL and NoSQL injection
- XSS (Cross-Site Scripting)
- Authentication bypasses
- Broken access control
- CSRF (Cross-Site Request Forgery)
- XXE (XML External Entities)
- SSRF (Server-Side Request Forgery)
- Insecure deserialization
- Path traversal
- Command injection

## Deployment

```bash
docker-compose up
```

Perfect for testing modern application security tools and learning about vulnerabilities in current web development stacks.
