Pixi is a vulnerable photo-sharing application created by the OWASP DevSlop project, built with the MEAN stack (MongoDB, Express, Angular, Node.js).

## Overview

Pixi demonstrates modern web application vulnerabilities in a microservices architecture, focusing on API security and contemporary development patterns.

## Technology Stack

- MongoDB: Database
- Express: Backend framework
- Angular: Frontend framework
- Node.js: Runtime environment

## Key Vulnerabilities

- SQL/NoSQL injection
- XSS (Cross-Site Scripting)
- Broken authentication
- Sensitive data exposure
- XXE (XML External Entities)
- Broken access control
- Security misconfiguration
- CSRF (Cross-Site Request Forgery)
- Insecure file upload
- API security issues

## Deployment

```bash
docker-compose up
```

## Learning Focus

- MEAN stack security
- Microservices vulnerabilities
- API security testing
- Modern JavaScript security

Ideal for developers and security professionals working with modern JavaScript stacks and microservices architectures.
