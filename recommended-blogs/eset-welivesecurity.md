## Why Follow This Blog

ESET's research team excels at analyzing the most sophisticated low-level threats, including UEFI rootkits and bootkits that most vendors struggle to detect. Their comprehensive coverage of mobile threats and unique regional visibility add further value.

## Key Topics Covered

### UEFI/Bootkit Research
- **UEFI Rootkits**: Firmware-level persistence
- **Bootkits**: Boot process manipulation
- **Firmware Threats**: Low-level malware analysis
- **Secure Boot Bypass**: Security mechanism defeats

### Mobile Security
- **Android Malware**: Comprehensive mobile threat coverage
- **Banking Trojans**: Mobile financial threats
- **Spyware**: Surveillance software analysis
- **App Store Threats**: Malicious application tracking

### APT Research
- **Nation-State Groups**: Detailed actor analysis
- **Campaign Tracking**: Operation documentation
- **Tool Analysis**: Custom malware examination
- **Regional Focus**: European threat landscape

### Traditional Malware
- **Ransomware**: Encryption malware analysis
- **Information Stealers**: Data theft malware
- **Backdoors**: Remote access threats
- **Botnet Operations**: Bot network analysis

## Pioneering UEFI Research

ESET discovered LoJax, the first in-the-wild UEFI rootkit, and has continued leading firmware threat research with investigations into MosaicRegressor, FinSpy UEFI capabilities, and various other UEFI implants.
