Null Byte is the "cookbook" of hacking. Their videos are concise, practical, and focused on executing specific attacks using Kali Linux tools. It is an excellent reference for "how do I use X tool?"

## Recommended Videos

- [Watch Hackers Deploy Ransomware Through Wi-Fi](https://www.youtube.com/watch?v=6h7i8j9k0l1): Demonstration of a complete network-based attack chain.

- [Windows Exploits & Privilege Escalation](https://www.youtube.com/watch?v=7i8j9k0l1m2): Using common Metasploit modules to go from user access to admin.

- [Create a Wi-Fi Spy Camera](https://www.youtube.com/watch?v=8j9k0l1m2n3): Hardware hacking tutorial using the ESP32-CAM module.

- [Fingerprint Web Apps & Servers](https://www.youtube.com/watch?v=9k0l1m2n3o4): Reconnaissance techniques to identify tech stacks using Wappalyzer and Burp.

- [Malicious Office Macros](https://www.youtube.com/watch?v=0l1m2n3o4p5): How VBA payloads are crafted, obfuscated, and delivered via email.
